# SpringBoot-Shiro-JWT-Scaffolding

## 介绍
这是一个基于SpringBoot-Shiro-JWT-Restful风格封装的脚手架模板

## 项目结构
aspect：AOP切面模块
common：公用工具模块  
mapper：mybatis相关模块  
model：实体模型  
service：业务逻辑模块  
shiro：鉴权模块  
util：项目专属工具类  
web：控制器相关模块
